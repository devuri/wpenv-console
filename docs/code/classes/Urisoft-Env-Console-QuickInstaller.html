generate_random_port()

Generate.php : 61

Generates a random port number within a specified range.

public generate_random_port([int $min = 5000 ][, int $max = 5100 ]) : int

This function generates a random port number using the mt_rand function. It accepts two optional parameters which define the minimum and maximum range for the random number. If the minimum value is greater than the maximum, the function swaps them to ensure the minimum is always less than the maximum.

Parameters
$min : int = 5000

Optional. The minimum value in the range. Default is 10010.

$max : int = 5100

Optional. The maximum value in the range. Default is 10900.

Tags
since
1.0.0
example

Usage

$randomPort1 = generateRandomPort(); // Default range $randomPort2 = generateRandomPort(10000, 10100); // Custom range

echo "Random Port Number (Default Range): " . $randomPort1 . "\n"; echo "Random Port Number (Custom Range): " . $randomPort2;
